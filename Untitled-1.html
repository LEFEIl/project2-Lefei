View peaks in individual graphs¶

What can peak data represent?

Answer:

Critical Points of Change: Peaks often indicate moments where the situation has reached a critical level. For example, a peak in 'unvaccinated' status might suggest a time when a large portion of a population was unvaccinated, potentially signaling increased risk for disease spread.

Public Health Policy: Peaks can influence public health policy decisions. If certain policies are followed by a decrease in peaks, they might be considered effective.

import pandas as pd

# Load your data
data = pd.read_csv('status death.csv')

# Convert 'Day' column to datetime
data['Day'] = pd.to_datetime(data['Day'])

# Find the peak (maximum) values for each status
peak_unvaccinated_value = data['unvaccinated'].max()
peak_vaccinated_with_value = data['vaccinated_with'].max()
peak_vaccinated_without_value = data['vaccinated_without'].max()

# Find the corresponding dates for these peak values
peak_unvaccinated_date = data[data['unvaccinated'] == peak_unvaccinated_value]['Day'].dt.strftime('%Y-%m-%d').iloc[0]
peak_vaccinated_with_date = data[data['vaccinated_with'] == peak_vaccinated_with_value]['Day'].dt.strftime('%Y-%m-%d').iloc[0]
peak_vaccinated_without_date = data[data['vaccinated_without'] == peak_vaccinated_without_value]['Day'].dt.strftime('%Y-%m-%d').iloc[0]

# Print the results
print("Peak Unvaccinated Date:", peak_unvaccinated_date)
print("Peak Vaccinated With Date:", peak_vaccinated_with_date)
print("Peak Vaccinated Without Date:", peak_vaccinated_without_date)
